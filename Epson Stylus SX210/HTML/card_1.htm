Introducing Memory Cards

Memory card copying precautions
You can use the printer’s memory card slots to copy files to a computer connected to the printer. You can also use it to copy files on your computer to your memory card.
For instructions on handling a memory card and printing from a memory card, see the booklet that came with your printer.

Memory card copying precautions

Keep the following in mind when copying files between your memory card and computer:
Check the documentation that came with your memory card and any adapter you may be using for any handling guidelines.
Do not eject your memory card or turn off the printer while the memory card light is flashing or you may lose data.
Do not copy files to a memory card while you are printing from the memory card.
The printer display does not update information about your memory card after you copy files to it or delete files from it. To update the information displayed, wait until the memory card light stops flashing, then eject and reload the memory card.
