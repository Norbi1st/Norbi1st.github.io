A nyomtatóillesztő eltávolítása

A nyomtatóillesztő eltávolítására a következő helyzetekben lehet szükség:
Frissíti az operációs rendszert a számítógépen.
Frissíti nyomtató illesztőprogramját egy újabb verzióval, kivéve, ha a frissítést a nyomtató-illesztőprogram frissítés funkciójával hajtja végre; a funkció ugyanis automatikusan eltávolít minden korábbi változatot.
Probléma van a nyomtatóillesztővel.
Ha újabb verzióra frissíti az operációs rendszert vagy a nyomtatóillesztőt, a frissítés, illetve az új illesztőprogram telepítése előtt feltétlenül távolítsa el az aktuális nyomtatóillesztőt. Ellenkező esetben nem fog működni az illesztőprogram frissítése.

Windows Vista esetében

Megjegyzés:
Windows Vista esetében a programok eltávolításához rendszergazda azonosítóra és jelszóra van szükség, amennyiben normál felhasználóként jelentkezik be.
Kapcsolja ki a nyomtatót, és húzza ki belőle az illesztőkábelt.
Kattintson az indítás gombra, és jelölje ki a Control Panel (Vezérlőpult) elemet.
A Programs (Programok) kategóriában kattintson az Uninstall a program (Program eltávolítása) elemre, majd válassza ki az EPSON Printer Software elemet a listából.
Kattintson az Uninstall/Change (Eltávolítás/módosítás) elemre, majd a Continue (Folytatás) elemre a User Account Control (Felhasználófiók kezelése) ablakban.
Válassza ki a nyomtató ikonját, majd kattintson az OK gombra.
A nyomtatóillesztő eltávolításához kattintson az OK gombra.

Windows XP és 2000 esetében

Megjegyzés:
Ahhoz, hogy Windows XP operációs rendszeren eltávolíthasson programokat, Computer Administrator (Számítógép adminisztrátor) jogokkal kell bejelentkeznie.
Windows 2000 rendszerben csak rendszergazdai jogokkal rendelkező (a rendszergazdák csoportjához tartozó) felhasználó fiókjával bejelentkezve távolíthat el programokat.
Kapcsolja ki a nyomtatót, és húzza ki belőle az illesztőkábelt.
Windows XP:
Kattintson a Start gombra, majd a Control Panel (Vezérlőpult) elemet.
Windows 2000:
Kattintson a Start menüre, majd a Settings (Beállítások) és a Control Panel (Vezérlőpult) elemre.
Kattintson kétszer az Add or Remove Programs (Programok hozzáadása és eltávolítása) ikonra, majd jelölje ki a lista EPSON Printer Software elemét.
Kattintson a Change/Remove (Módosítás/eltávolítás) elemre.
Válassza ki a nyomtató ikonját, majd kattintson az OK gombra.
A nyomtatóillesztő eltávolításához kattintson az OK gombra.

Mac OS X esetében

Megjegyzés:
Csak Computer Administrator (Számítógép adminisztrátor) jogokkal bejelentkezve lehet programokat eltávolítani. Korlátozott jogokkal rendelkező felhasználói fiókkal nem lehet programokat eltávolítani.
Zárjon be minden futó alkalmazást.
Helyezze be az EPSON Printer Software CD-lemezét a Macintosh.
Kattintson kétszer a Mac OS X ikonra az Epson mappában. Megjelenik a CD Installer (CD telepítő) képernyő. Kattintson a Custom Install (Egyéni telepítés) elemre.
Válassza ki a Printer Driver (Nyomtató meghajtó) elemet.
Megjegyzés:
Ha megjelenik az Engedélyezés párbeszédpanel, adja meg a jelszót, majd kattintson az OK gombra.
Olvassa el a licencszerződést, majd kattintson az Accept (Elfogad) gombra.
Válassza a menüben baloldalon felül található Uninstall (Eltávolítás) elemet, majd kattintson az Uninstall (Eltávolítás) gombra.
Az illesztőprogram eltávolítását a képernyőn megjelenő utasításokat követve végezheti el.
Megjegyzés Mac OS X 10.3 vagy 10.4 felhasználóknak:
Ha a nyomtató neve megmarad a Print Center (Nyomtatóközpont) alkalmazásban vagy a Printer Setup Utility (Nyomtatóbeállítási segédprogram) panelen, válassza ki a nevet a Printer List (Nyomtatólista) alatt, és kattintson a Delete (Törlés) gombra.

Egyéb Epson alkalmazások eltávolítása

Eltávolíthatja azokat az alkalmazásokat, amelyekre nincs szüksége. Hajtsa végre az alábbi lépéseket.

Windows Vista esetében

Megjegyzés:
Windows Vista esetében a programok eltávolításához rendszergazda azonosítóra és jelszóra van szükség, amennyiben normál felhasználóként jelentkezik be.
Kattintson az indítás gombra, és jelölje ki a Control Panel (Vezérlőpult) elemet.
A Programs (Programok) kategóriában kattintson az Uninstall a program (Program eltávolítása) lehetőségre.
A programok listájából jelölje ki az eltávolítani kívánt szoftvereket.
Kattintson az Uninstall/Change (Eltávolítás/módosítás) elemre, majd a Continue (Folytatás) elemre a User Account Control (Felhasználófiók kezelése) ablakban.
A megerősítést kérő üzenetnél kattintson a Yes (Igen) gombra.
Kövesse a képernyőn megjelenő utasításokat.

Windows XP és 2000 esetében

Megjegyzés:
Ahhoz, hogy Windows XP operációs rendszerből alkalmazásokat távolíthasson el, Computer Administrator (Számítógép adminisztrátor) jogosultságokkal kell bejelentkeznie. Korlátozott jogokkal rendelkező felhasználói fiókkal nem lehet programokat eltávolítani.
Windows 2000 rendszerben csak adminsztrátori jogokkal rendelkező (a rendszergazdák csoportjához tartozó) felhasználó fiókjával bejelentkezve távolíthat el alkalmazásokat.
Windows XP:
Kattintson a Start gombra, majd a Control Panel (Vezérlőpult) elemet.
Windows 2000:
Kattintson a Start menüre, majd a Settings (Beállítások) és a Control Panel (Vezérlőpult) elemre.
Kattintson duplán az Add or Remove Programs (Programok hozzáadása és eltávolítása) ikonra.
A programok listájából jelölje ki az eltávolítani kívánt szoftvereket.
Kattintson a Change/Remove (Módosítás/eltávolítás) elemre.
A megerősítést kérő üzenetnél kattintson a Yes (Igen) gombra.
Kövesse a képernyőn megjelenő utasításokat.

Mac OS X esetében

Megjegyzés:
Csak Computer Administrator (Számítógép adminisztrátor) jogokkal bejelentkezve lehet alkalmazásokat eltávolítani. Korlátozott jogokkal rendelkező felhasználói fiókkal nem lehet programokat eltávolítani.
Az alkalmazástól függően előfordulhat, hogy a telepítőprogram és az eltávolítóprogram nem ugyanaz.
Zárjon be minden futó alkalmazást.
Helyezze be az EPSON Printer Software CD-lemezét a Macintosh.
Kattintson kétszer a Mac OS X ikonra az Epson mappában. Megjelenik a CD Installer (CD telepítő) képernyő. Kattintson a Custom Install (Egyéni telepítés) elemre.
Válassza ki az eltávolítandó alkalmazást.
Ha megjelenik a licencszerződés, olvassa el, majd kattintson az Accept (Elfogad) gombra.
Válassza a menüben baloldalon felül található Uninstall (Eltávolítás) elemet, majd kattintson az Uninstall (Eltávolítás) gombra.
Az alkalmazás eltávolítását a képernyőn megjelenő utasításokat követve végezheti el.
Megjegyzés:
Ha nem választotta ki az Uninstall (Eltávolítás) elemet a 6. lépésben, duplán kattintson azApplications (Programok) mappára a Mac OS X merevlemezén, válassza ki az eltávolítandó alkalmazást, majd húzza át a Trash (Kuka) ikonba.
