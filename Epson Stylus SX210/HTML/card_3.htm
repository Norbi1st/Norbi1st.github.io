Removing the Memory Card

For Windows
Caution:
Do not remove a memory card when the memory card light is flashing or you may lose data.
Make sure the memory card light is not flashing.
Pull the memory card out of its slot.
For Mac OS X
Caution:
Do not remove a memory card when the memory card light is flashing or you may lose data.
Make sure the memory card light is not flashing.
Drag the removable disk icon () on your desktop into the trash.
Caution:
Always drag the removable disk icon on your desktop into the trash before you remove the card or you may lose data on the card.
Pull the memory card out of its slot.
